# 專案（部分）

  • SQLab/CRAXplusplus: CRAX 是交大 SQLab 發展的脅迫（Exploit）產生器，而 CRAXplusplus 的設計理念是成為一個系統架構清晰、易於擴充、能夠復現實驗、且能於將來整合多篇 AEG 研究的半自動脅迫產生框架。給定一個 x86_64 Linux ELF 與概念測資，該輸入會構成一條特定的執行路徑，本系統運用擬真執行（Concolic Execution）來搜集該路徑的限制式，並且對其添加脅迫限制式，再透過 SMT 求解器計算出最終的脅迫，並進一步後製成脅迫腳本。此外，本系統允許使用者新增自訂的脅迫技術模型（Technique）與分析模組（Module），以期將系統的可擴充性最大化。

  • valkyrie: 針對 AArch64 撰寫的 UNIX-like 作業系統，包含：佔先式多執行序（kernel/user）、寫入時複製（CoW） fork() 、支援虛擬記憶體、VFS、FAT32、Dev/Proc/Tmp 檔案系統，以及一個自幹的 C++20 標準函示庫。目前僅可於 QEMU 運行起來，自從不知道哪個功能寫完後放到板子上會撞到 vtable 與 memory alignment 的問題。

  • wmderland: 使用空間分割樹的 X11 平鋪式視窗管理器，可理解成簡化版的 i3wm。

  • vigilante: 使用 cocos2d-x 與 C++17 撰寫的像素風的 2D 遊戲，類似惡魔城，但玩家能夠招募 NPC 作為隊友。此遊戲包含完整的戰鬥、裝備、任務、交易、組隊系統，但因為工作太忙了，沒時間將劇情、音樂補齊。

  • hacklympics: 在軍校時寫的一個 OJ：前端 JavaFX，後端 Django，事件驅動，主要特色是教師身分的使用者可以監考。學生寫 code 的 TextArea 會定期回傳 diff 回 server，再 broadcast 到所有教師的 client 並套用 diff。
